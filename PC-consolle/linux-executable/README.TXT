Lcss_adapter_consolle v1.0 has been tested on Lubuntu 18.04

INSTALLATION:
1) download the file: lcss_adapter_consolle_linux_executable.zip and decompress in your favourite folder (e.g: /home/user)

USE:
1) open the terminal, change the working directory in the one where there is the lcss_adapter_consolle (e.g. /home/user)
2) launch the program by typing in the terminal "./lcss_adapter_consolle"
